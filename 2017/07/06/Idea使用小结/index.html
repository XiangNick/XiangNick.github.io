Idea使用小结

2017-07-06

一、初始配置

  1. 配置jdk
    安装完毕后，选择Configure->Project Defaults->Project Structure进行jdk的设置

在这两处设置同样的版本(1.7对应7)

如果此处显示的为No SDK,则需要去SKDs里配置

点击+号，选择jdk的路径

设置完毕后点击Apply

这样在上一步的jdk就可以设置为现在新增的。

  1. 配置maven

点击Project Defaults->Settings

在搜索框中搜索 maven
在maven的设置界面，点击右方的小按钮，选择maven的路径

User settings file和Local repository保持默认的即可。

  1. 配置tomcat

创建一个maven项目并启动idea。进入代码界面后，点击上方配置按钮配置tomcat

点击+，找到Tomcat Server节点，选择Local，

配置解压好的Tomcat

可以配置tomcat的名字和端口号。

配置完毕后点击Deployment来配置war包解压路径。

点击右侧+号，选择Artifact，将我们创建好的项目的war包部署在tomcat的根目录/

二、其他配置

tool-buttons需要打开

否则右侧快捷按钮将不显示

在idea新建maven项目记得勾选Create from archetype，
在使用webapp这个骨架创建的maven项目，初始结构如下

需要初始化目录结构。

按如下目录结构新建文件夹

  • 在main目录下新建java目录(右键Mark Directory As->Source Root)
  • 在src目录下(和main同级)新建test目录
  • 在test目录下新建java目录(右键Mark Directory As-> Test Source Root)
  • 把main下的resources文件夹(右键Mark Directory As->Resources Root)

tips:
将xml等资源文件放置到main下的resources文件夹中(别忘了标注为Resources Root)。这样maven工具默认在编译的时候，会将resources文件夹中的资源文件一块打包进classes目录中，否则打出的war包不会有非.java文件，并且那些需要classpath:的配置文件也无法找到路径。(即标注了后，resources目录就是classpath的根目录)

配置properties支持ascii编码

三、idea配置jrebel热部署

jrebel是非常方便的热部署插件，修改java代码和配置文件，不需要重启web容器即可访问修改内容。

3.1 安装Jrebel插件

File->Settings->Plugins->Browse respositories
搜索Jrebel for Intellij

安装完毕后重启idea

3.2 激活Jrebel插件

方式一：MyJrebel邀请码永久激活

MyJrebel
需要翻墙，访问该网站，使用推特或者facebook帐号登录即可。



获取该注册码，进入重启后的idea

Help -> Jrebel -> Activation

选择I already have a licensing，将刚才的邀请码填入

即可激活完毕。

方式二：使用破解补丁或者设置server

百度即可

3.3 配置热部署

先配置tomcat

① 点击idea中tomcat配置

② 配置deployment为exploded

③ 配置server，将 “On Update action”、”On frame deactivation” 都选择 update classes and resources

再配置Jrebel

④view -> tool windows -> Jrebel

将项目勾选上

3.4 使用热部署

选择tomcat，使用Jrebel的debug模式启动即可！

任意修改 java 代码，并 ctrl + s 保存 ，update classes and resources 一下。
（如果配置了idea的自动编译，这一步都不用，修改完代码直接切到浏览器即可）

四、快捷键

快捷键

  • Alt+回车 导入包,自动修正
  • Ctrl+N 查找类
  • Ctrl+Shift+N 查找文件
  • Ctrl+Alt+L 格式化代码
  • Ctrl+Alt+O 优化导入的类和包
  • Alt+Insert 生成代码(如get,set方法,构造函数等)
  • Ctrl+E或者Alt+Shift+C 最近更改的代码
  • Ctrl+R 替换文本
  • Ctrl+F 查找文本
  • Ctrl+Shift+Space 自动补全代码
  • Ctrl+空格 代码提示
  • Ctrl+Alt+Space 类名或接口名提示
  • Ctrl+P 方法参数提示
  • Ctrl+Shift+Alt+N 查找类中的方法或变量
  • Alt+Shift+C 对比最近修改的代码
  • Shift+F6 重构-重命名
  • Ctrl+Shift+先上键
  • Ctrl+X 删除行
  • Ctrl+D 复制行
  • Ctrl+/ 或 Ctrl+Shift+/ 注释（// 或者/…/ ）
  • Ctrl+J 自动代码
  • Ctrl+E 最近打开的文件
  • Ctrl+H 显示类结构图
  • Ctrl+Q 显示注释文档
  • Alt+F1 查找代码所在位置
  • Alt+1 快速打开或隐藏工程面板
  • Ctrl+Alt+ left/right 返回至上次浏览的位置
  • Alt+ left/right 切换代码视图
  • Alt+ Up/Down 在方法间快速移动定位
  • Ctrl+Shift+Up/Down 代码向上/下移动。
  • F2 或Shift+F2 高亮错误或警告快速定位
    代码标签输入完成后，按Tab，生成代码。
    选中文本，按Ctrl+Shift+F7 ，高亮显示所有该文本，按Esc高亮消失。
  • Ctrl+W 选中代码，连续按会有其他效果
    选中文本，按Alt+F3 ，逐个往下查找相同文本，并高亮显示。
  • Ctrl+Up/Down 光标跳转到第一行或最后一行下
  • Ctrl+B 快速打开光标处的类或方法

    Intellij IDEA最常用快捷键

  1. Ctrl＋E，可以显示最近编辑的文件列表
    2 .Shift＋Click可以关闭文件
  2. Ctrl＋[或]可以跳到大括号的开头结尾
  3. Ctrl＋Shift＋Backspace可以跳转到上次编辑的地方
  4. Ctrl＋F12，可以显示当前文件的结构
  5. Ctrl＋F7可以查询当前元素在当前文件中的引用，然后按F3可以选择
  6. Ctrl＋N，可以快速打开类
  7. Ctrl＋Shift＋N，可以快速打开文件
  8. Alt＋Q可以看到当前方法的声明
  9. Ctrl＋W可以选择单词继而语句继而行继而函数
  10. Alt＋F1可以将正在编辑的元素在各个面板中定位
  11. Ctrl＋P，可以显示参数信息
  12. Ctrl＋Shift＋Insert可以选择剪贴板内容并插入
  13. Alt＋Insert可以生成构造器/Getter/Setter等
  14. Ctrl＋Alt＋V 可以引入变量。例如把括号内的SQL赋成一个变量
  15. Ctrl＋Alt＋T可以把代码包在一块内，例如try/catch
  16. Alt＋Up and Alt＋Down可在方法间快速移动
